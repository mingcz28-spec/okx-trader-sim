# OKX Trader Sim

一个用于 OKX 合约模拟盘的 Web 控制台骨架。

## 当前状态

- Next.js 前端已搭好
- 已有前后端联动
- 已支持本地模拟状态读取
- 已支持保存 API 配置、更新风控、模拟开仓、全部平仓
- 已支持通过 OKX API 同步真实账户与模拟盘的资金、持仓信息
- 已支持资金币种明细与最近订单历史展示
- 真实盘金额显示默认按 CAD 换算
- 当前数据保存在服务端内存，重启后会重置

## 下一步

1. 把 API Key 改成安全环境变量或密钥存储
2. 增加真实 OKX 下单、平仓接口（建议先只做模拟盘）
3. 增加成交历史、资金曲线、按天收益
4. 增加策略执行器和回测模块
5. 增加真实风控逻辑和交易审计日志

## 本地运行

```bash
npm install
npm run dev
```

然后打开 <http://localhost:3000>

## 风险提醒

自动合约交易风险很高，没有任何策略能保证 100% 收益。
先用模拟盘，先测风控，再考虑实盘。
